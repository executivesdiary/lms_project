Lead Management System

A custom-built Django application powering outreach and editorial workflow at Executives Diary.

Built for Community Builders, Editors, Project Managers, and Super Admins.

✅ What We’ve Built So Far

  • 🔐 Multi-role user system (Super Admin, Project Manager, Community Builder, Editor)
  • 📬 Community Builders can add leads, convert them into connections, and update statuses
  • 📈 Builder dashboards show analytics and outreach progress (past 30 days)
  • 📎 Screenshot uploads and comment threads on each connection
  • 🧠 Project Managers can assign Editors and monitor builder activity
  • ✍️ Editors view assigned leads and track biography work
  • 📊 Analytics and pie charts for status insights
  • 📌 Threaded AJAX-powered comment system between team members

🚀 Upcoming Features

  • 📥 LinkedIn CSV import to auto-populate leads
  • 🧹 AJAX-powered lead cleanup and filtering
  • 🤖 Integrated biography writing assistant using GPT
  • 📨 Email sequence automation for cold leads
  • 📁 Export connections as Excel/PDF reports
  • 💬 Real-time chat between editors and builders (optional future)
  • 🧪 Role-based testing dashboard with demo data for advisors
